Psuedo Code Examples:

Temperature Conversion:
    fahrenheit <-- prompt user for temp in fahrenheit
    celsius <----- (5/9)*(fahrenheit - 32)
    print temperature in celsius


Child Tax Credit

Cookie Monster
    cookies = prompt user for number of cookies
    while number of cookies is less than 4 prompt the user for a number of cookies
    after a number >= 4 is entered print out thank you message.
